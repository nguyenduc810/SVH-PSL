# SVH-PSL
The source code for the paper: **Improving Pareto Set Learning for Expensive Multi-objective Optimization via Stein Variational Hypernetworks** ([Link](https://arxiv.org/abs/2412.17312)) has been accepted to AAAI 2025.

# Run experiment
1. Create a new environment following requirements.txt

2. Execute the following files to obtain the results: run_svh.py, run_psl_mobo.py, and run_DA.py
